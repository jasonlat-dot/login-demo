# 高颜值登录注册切换页面

基于 **Vue3 + Element Plus** 的轻奢商务风格登录注册切换页，零额外框架依赖。

## 特性

- 全屏动态粒子背景：原生 Canvas 实现，粒子随机漂浮、靠近连线、鼠标排斥交互
- 多层光晕流光渐变底色
- 左右分栏：左侧品牌宣传 + 右侧毛玻璃卡片
- 登录/注册面板平滑淡入淡出 + 位移 + 缩放切换
- 完整表单实时校验（手机号、密码强度、确认密码、协议）
- 密码框内置眼睛图标，可切换明文/密文
- 卡片边角旋转光晕、输入框聚焦外发光、按钮按压反馈
- 提交带 loading 防重复，成功弹出漂亮提示弹窗

## 启动

```bash
npm install
npm run dev
```

打开浏览器访问 `http://localhost:5173`。
